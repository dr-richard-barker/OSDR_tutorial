# OSDR Metadata Curation Guidelines Summary

* **For GEO-sourced data**:
  * Only list "contributors" in the people section.
  * Use the author ontology if there's an associated publication.
  * If the publication is too broad or lists multiple GSEs, opt for the GEO summary as the description.
  * Enter full funding statements in the funding field, not just grant numbers.
* **General Rules**:
  * Maintain the same sample order as in NCBI.
  * Exclude special characters from factor levels; use only letters, numbers, and spaces.
  * Retype any special characters from publications, substituting mu (Î¼) with "u".
  * For associated publications, use keywords as study design descriptors.
* **Excel Handling**:
  * If editing outside GEODE, remove blank columns at the end before re-importing.
* **Formatting**:
  * Prioritize characteristics, then factors, then protocols and parameters in column order in the sample table. Comments positioning is flexible.



**Metadata normalization:**

**\*Partial Weight Bearing** is referring to the quadrupedal unloading model in rats developed by Marie Morteux et al., described in [this manuscript](https://journals.physiology.org/doi/full/10.1152/japplphysiol.01083.2017). The values of this column would be _normal loading with full harness (PWB100) for controls,_ _40% of normal loading (PWB40)_ for Mars gravity simulation, 2_0% of normal loading (PWB40)_ for Moon gravity simulation, or 7_0% of normal loading (PWB40)._

\


\*\*In GLDS-208 the author uses Root Zone to refer to distinct sections of the primary root. In plant studies, the 'Root Zone' is also commonly used to describe the entire rhizosphere, which would include the soil (or agar) around the roots as well as the root tissue itself.&#x20;

\*\*\*Tissue is considered as a factor only if it was an intended factor and direct comparisons like DGE are done with tissue types. If they don't meet this criteria, the dataset should be split into separate studies by tissue type (even if they are from the same accession number from an outside repository).

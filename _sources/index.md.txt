# Welcome to Open Science Data Repository (OSDR) tutorial documentation!

OSDR is a public science data library for space biology researchers and future space explorers. It pulls data from the [OSDR NASA Bio](https://osdr.nasa.gov/bio/) and offers a simple and intuitive API.


> **Note:**
> This project is under active development v2.
